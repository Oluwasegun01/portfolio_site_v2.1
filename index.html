Projects

Multi-Tier Web Application Deployment on AWS (Lift & Shift Strategy)

Deployed a multi-tier web application on AWS using a Lift & Shift approach to replicate an on-premise stack in the cloud. Set up and configured Apache Tomcat (application layer), MySQL (database), Memcached (caching), and RabbitMQ (messaging queue) across separate EC2 instances. Ensured secure communication between tiers using VPC, security groups, and IAM roles. This project strengthened my understanding of cloud infrastructure, scalability, and service isolation in a production-like environment.

Application Refactoring with AWS Managed Services

Refactored a traditional multi-tier web application to use AWS managed services for improved scalability, availability, and operational efficiency. Deployed the application using AWS Elastic Beanstalk, migrated the database to Amazon RDS, implemented Amazon ElastiCache for caching, and replaced the messaginglayer with Amazon MQ. This project deepened my understanding of cloud-native architecture, service integration, and infrastructure automation.

End-to-End CI/CD Pipeline with Jenkins and AWS ECS

Built a complete CI/CD pipeline using Jenkins to automate the build, test, and deployment process for a Java-based application. Integrated Maven for build automation, Checkstyle and SonarQube for code quality analysis, and Nexus for artifact storage. Containerized the application using Docker, pushed images to AWS Elastic Container Registry (ECR), and deployed to Elastic Container Service (ECS). This project improved my hands-on skills in pipeline orchestration, containerization, and cloud-native deployments.

CI/CD Pipeline with GitHub Actions and AWS ECS

Implemented a CI/CD pipeline using GitHub Actions to automate the build, test, and deployment workflow for a Java application. Utilized Maven for build automation, integrated Checkstyle and SonarCloud for static code analysis and quality checks. Containerized the application with Docker, published images to AWS Elastic Container Registry (ECR), and deployed to Elastic Container Service (ECS). This project enhanced my understanding of GitHub-native automation, code quality enforcement, and cloud-based deployments.
